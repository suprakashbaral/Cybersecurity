# Day 03 Lab

## Task 1

Identify the current user.

```bash
whoami
```

---

## Task 2

Display your UID and groups.

```bash
id
```

---

## Task 3

List all groups.

```bash
groups
```

---

## Task 4

Display all logged-in users.

```bash
who
```

---

## Task 5

Show active users.

```bash
w
```

---

## Task 6

View the passwd file.

```bash
cat /etc/passwd
```

---

## Task 7

Find your username in passwd.

```bash
grep $USER /etc/passwd
```

## Task 8

View the group file.

```bash
cat /etc/group
```
