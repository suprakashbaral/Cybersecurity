# Day 1 - Linux Commands

## Commands Learned

### pwd

```bash
pwd
```

Purpose:
Shows the current working directory.

---

### ls

```bash
ls
```

Purpose:
Lists files and folders.

---

### ls -l

```bash
ls -l
```

Purpose:
Shows files in long format.

---

### ls -la

```bash
ls -la
```

Purpose:
Shows all files, including hidden files.

---

### whoami

```bash
whoami
```

Purpose:
Displays the current logged-in user.

---

### clear

```bash
clear
```

Purpose:
Clears the terminal screen.

---

### mkdir

```bash
mkdir practice
```

Purpose:
Creates a new directory.

---

### touch

```bash
touch file.txt
```

Purpose:
Creates an empty file.
