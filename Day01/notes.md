# Day 1 Notes

# What is Linux?

Linux is a free and open-source operating system (OS) based on the Unix operating system. It acts as a bridge between computer hardware and software, allowing users to manage files, run programs, and control system resources efficiently.

Unlike Windows or macOS, Linux is developed by a global community of developers, making it highly secure, stable, and customizable. It is widely used in servers, cloud computing, cybersecurity, embedded systems, and supercomputers.

---

# History of Linux

- In 1991, Linus Torvalds, a computer science student at the University of Helsinki, Finland, started developing Linux as a personal project.
- He wanted a free operating system similar to UNIX.
- On **25 August 1991**, Linus announced his project on the Minix newsgroup.
- The first Linux kernel (version 0.01) was released in **September 1991**.
- Developers around the world contributed to the project, making Linux one of the largest open-source software projects.
- Today, Linux powers millions of servers, smartphones (Android), cloud platforms, IoT devices, and supercomputers.

---

# Why Learn Linux for Cybersecurity?

Linux is one of the most important operating systems for cybersecurity professionals because:

- Most web servers run Linux.
- Popular penetration testing distributions like Kali Linux are based on Linux.
- Security tools such as Nmap, Metasploit, Wireshark, Burp Suite, and John the Ripper work best on Linux.
- Linux provides powerful command-line tools for system administration and automation.
- Digital forensics and incident response often require Linux knowledge.

# Commands Learned Today

| Command | Purpose |
|---------|---------|
| `pwd` | Displays the current working directory |
| `ls` | Lists files and folders |
| `ls -l` | Lists files in detailed format |
| `ls -la` | Lists all files, including hidden files |
| `cd` | Changes the current directory |
| `mkdir` | Creates a new directory |
| `touch` | Creates a new file |
| `whoami` | Displays the current logged-in user |
| `clear` | Clears the terminal screen |
